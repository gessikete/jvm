==============================================================

Descrição: Este programa tem como objetivo simular uma JVM. Nessa primeira etapa
simularemos um leitor e exibidor de um arquivo .class

==============================================================

ATENÇÃO: Este programa foi desenvolvido e testado nas plataformas windows e linux

==============================================================

COMPILAÇÃO
Para compilar esse programa utilizando o terminal a partir da raiz do projeto execute:
$ make

EXECUÇÃO
Para executar esse programa utilizando o terminal a partir da raiz do projeto execute:
$ ./jvm.exe { -e | -l } {arquivo.class}

De forma que:
{ -e | -l }
-e caso queira carregar o arquivo .class e escrever a saída no arquivo saida.txt
-l caso queira somente carregar o arquivo .class
